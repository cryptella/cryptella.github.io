Please note the volume.

Cursor key Character selection and others.
Space key or Z key Decision and change of each value.
Shift key or X key Returns to a character selection screen,
and a set screen.
R, T key BGM switch.
Q, W key Background switch.
E key Visible of window and panel On/Off.
A, S, D key Image quality change.

Please see 'Animetica explanation' of the Manual about this detailed explanation.
